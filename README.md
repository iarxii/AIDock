# AIDock

AIDock is a powerful, lightweight Docker orchestration stack designed to make running local AI workflows completely accessible and foolproof. It dynamically manages local LLM models and automatically organizes your AI-generated project files.

## Features

- 🚀 **Interactive Setup**: Simple CLI to configure your workspace and select AI profiles.
- 📂 **Organized Workspaces**: Files generated by the AI are cleanly isolated into sub-folders within your chosen root directory.
- 🧠 **Persistent Memory**: Powered by Postgres and `pgvector`, your agent remembers previous conversations and code context even after reboots.
- 🧹 **On-Demand Models**: Pulls only the AI model you need for your specific task profile, with easy pruning commands to save disk space.

## Prerequisites

- **Docker Desktop** (version 4.41+ for Windows, 4.40+ for macOS/Linux) installed and running.
  > [!IMPORTANT]
  > **Enable Docker Model Runner**: Before attempting to start the workflow, you must explicitly enable the Docker Model Runner to ensure model pull commands do not fail.
  > 1. Open **Docker Desktop**.
  > 2. Click the **Settings** (gear icon) at the top.
  > 3. Navigate to the **AI** tab on the left menu.
  > 4. Check **Enable Docker Model Runner**.
  > 5. Click **Apply & restart**.
- **Python 3.10+** (for running the setup CLI).

## Quickstart

1. **Run Interactive Setup**
   This will automatically create an isolated environment, install dependencies, check Docker, prompt you for a workspace path, and let you select a model profile.
   
   **Windows:**
   ```cmd
   aidock.bat setup
   ```
   **Mac/Linux:**
   ```bash
   ./aidock.sh setup
   ```

2. **Start the Stack**
   **Windows:** `aidock.bat start`
   **Mac/Linux:** `./aidock.sh start`

3. **Access the UI**
   Open your browser to `http://localhost:3000`.

## Available Commands

- `aidock setup` - Configure workspace and model profile.
- `aidock start` - Start all containers.
- `aidock stop` - Stop all containers.
- `aidock prune` - Remove unused Docker images to free up space.
