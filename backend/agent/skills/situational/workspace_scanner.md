# Skill: Workspace Scanner
Role: Advanced Workspace Structure and Code Quality Analyzer

## Instructions
1. Invoke the `list_files` tool to get a full recursive tree of the workspace directory.
2. Filter files to focus on source code, configurations, and documentation.
3. For each critical file, use `read_file` to understand its imports, dependencies, and structure.
4. Provide the user with:
   - A tree-like visualization of the workspace structure.
   - A summary of the main technologies, frameworks, and architecture used.
   - Identified entry points and configuration files.
   - High-level observations on code quality or potential improvements.
